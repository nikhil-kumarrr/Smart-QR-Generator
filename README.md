# Custom QR Generator
A modern, lightweight QR Code Generator built with Python and Streamlit.
Generate clean, scannable QR codes instantly with a minimal and professional UI.

##  Features
- ⚡ Instant QR generation
- 🎯 Centered, clean UI layout
- 🎨 Custom styled interface (CSS + Streamlit)
- ⬇️ One-click PNG download
- 🔒 URL validation
- 📱 Responsive design

## 🛠 Tech Stack
- Python 3.x
- Streamlit
- Pillow (PIL)
- Custom CSS for UI

## 📦 Installation & Setup
### 1️⃣ Clone the repository
```
git clone https://github.com/your-username/custom-qr-generator.git
cd custom-qr-generator
```

### 2️⃣ Create virtual environment
```
2️⃣ Create virtual environment
```

### 3️⃣ Activate environment
#### Window :
```
venv\Scripts\activate
```
#### Mac/Linux:
```
source venv/bin/activate
```
### 4️⃣ Install dependencies
```
pip install -r requirements.txt
```

### 5️⃣ Run the app
```
streamlit run app.py
```
